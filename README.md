# Postman API Counter

### How to use
- Download `postman.py`
- Run this to count the API
```bash
python postman.py ~/path/to/postman_collection.json
```
- To count spesific folder (1 depth only)
```bash
python postman.py ~/path/to/postman_collection.json FolderName
```
- Help menu
```bash
python postman.py -h
```
